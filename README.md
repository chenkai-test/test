# Java Learning Project

My first GitHub repository - Java learning project.

## Tech Stack

- Java
- JUnit 4
- Git / GitHub

## Project Structure

```
src/
├── main/java/com/chenkai/
│   ├── HelloWorld.java      # Hello World program
│   └── Calculator.java      # Simple calculator
└── test/java/com/chenkai/
    └── CalculatorTest.java   # Calculator unit tests
```

## How to Run

```bash
# Compile
javac src/main/java/com/chenkai/HelloWorld.java

# Run
java -cp src/main/java com.chenkai.HelloWorld
```
